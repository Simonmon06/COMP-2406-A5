Hosting A5 Server On Heroku

Here is a useful set of instructions that Omar Garcia Flores has sent me that he used to host his assignment 5 server on Heroku -Thanks Omar.

 

These are the steps I followed to set up the tutorial 9/assignment 5 demo code on a Heroku server.

  1. Set up a GitHub repository 
    https://help.github.com/articles/create-a-repo/
  2. Install Heroku locally 
    https://devcenter.heroku.com/articles/heroku-cli#download-and-install
  3. Locally, on the repository’s directory, generate a Heroku app remote

$heroku create
https://devcenter.heroku.com/articles/git

  1. Push changes to the Heroku remote 
    $git push heroku master
  2. Create an account on mLab (MongoDB tool for cloud storage)
    https://mlab.com/signup/
  3. Create a new database on mLab and connect the app to mLab using the given connection string similar to: 
    `mongodb://<dbuser>:<dbpassword>@ds038888.mlab.com:38888/<databaseName> `
  4. Replace the line  
    `mongoose.connect('mongodb://localhost/myShoppingApp3')`
    with 
    `mongodb://<dbuser>:<dbpassword>@ds038888.mlab.com:38888/<databaseName>`
    in both populate-for-startup.js and app.js
  5. Run populate-for-startup.js locally, this should populate the databse in the cloud (mLab database).
  6. Upload changes to git repository and Heroku app
    $git add .
    $git commit -m “Changes”
    $git push 
    //Updates heroku app
    $git push Heroku master
